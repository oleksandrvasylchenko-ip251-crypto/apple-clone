# 🍎 Apple Clone - Production Ready

A modern Apple website clone with responsive design, smooth animations, and dark mode.

## ✨ Features

- ✅ Hero sections with GSAP animations
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Dark mode (Press D)
- ✅ Video slider with product images
- ✅ Mobile burger menu
- ✅ Smooth scrolling
- ✅ Product showcase grid
- ✅ Technology ecosystem display

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm start

# Build for production
npm run build
```

## 🎮 Controls

- **D** - Toggle dark mode
- **← →** - Navigate video slider
- **Click dots** - Jump to slide

## 📱 Responsive

- Desktop (1024px+)
- Tablet (768px - 1024px)
- Mobile (480px - 768px)
- Small Mobile (<480px)

## 🛠️ Tech Stack

- HTML5
- SCSS
- TypeScript
- GSAP
- Three.js
- Webpack 5

## 📦 Build

Production build creates optimized `dist/` folder with:
- Minified code
- Optimized assets
- Ready for deployment

## 🌐 Deploy

Upload `dist/` folder to any hosting service.
